Release notes

Drazlo 1.0.13

July 7, 2026

Added

  • Editor trim: After recording, drag in/out handles to trim the clip. Preview and export use the trimmed range. Captions outside the trim are excluded from export (sidecar unchanged).

Fixed

  • Free layout: Selecting 1–4 windows now seeds to a clear grid — top left, top right, bottom left, bottom right — instead of overlapping in the center.

Changed

  • Trim UI: New range slider with primary accent styling (replaces the yellow timeline bar).

If in-app update fails, download manually: drazlo.vercel.app/download
